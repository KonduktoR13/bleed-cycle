# Тихий цикл

PWA-календарь эмоционального фона по фазам менструального цикла.

Постоянная версия после публикации GitHub Pages:

https://konduktor13.github.io/bleed-cycle/

Локальный запуск для установки или проверки:

```bash
python3 start_server.py
```

Проверка расчетной модели:

```bash
node tests/model.test.js
```
